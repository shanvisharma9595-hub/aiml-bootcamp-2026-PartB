# 🐍 Python Fundamentals Practice

## 📌 Overview

This repository includes Python practice programs focused on understanding and implementing core programming concepts. The exercises demonstrate basic data handling, decision-making, and simple logic building using Python.

## 🛠️ Tools & Technologies

- Python 3
- Google Colab

## 📚 Concepts Implemented

### 1. Lists
- Created and worked with lists to store multiple values.
- Practiced accessing list elements and iterating through data using loops.

### 2. Dictionaries
- Used dictionaries to represent structured data in key-value format.
- Added, accessed, and modified dictionary values.

### 3. Iteration Using Loops
- Applied `for` loops to process collections like lists and dictionaries.
- Used loops for traversing and displaying stored information.

### 4. Decision Making
- Implemented conditional logic using:
  - `if`
  - `elif`
  - `else`
- Created programs based on different conditions and outcomes.

### 5. Data Processing Practice
- Performed basic analysis on student information using logical conditions.
- Checked different parameters such as:
  - Performance level
  - Attendance criteria
  - Assignment completion status

## 🎯 Learning Objective

The main goal of this repository is to strengthen Python programming fundamentals and improve logical thinking through practical coding exercises.
